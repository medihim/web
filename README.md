# 메디힘 홈페이지

메디힘 공식 홈페이지의 GitHub Pages 배포용 정적 HTML 패키지입니다.

## 배포 방법

1. GitHub에서 새 저장소를 생성합니다.
2. 이 폴더 안의 파일과 폴더를 저장소 최상위 경로에 업로드합니다.
3. 저장소의 **Settings → Pages**로 이동합니다.
4. **Build and deployment**에서 `Deploy from a branch`를 선택합니다.
5. Branch는 `main`, Folder는 `/(root)`를 선택하고 저장합니다.
6. 배포가 완료되면 GitHub Pages 주소에서 홈페이지를 확인합니다.

## 시작 파일

- `index.html`: 메인 홈페이지
- `review.html`: 내부 검수용 페이지
- `assets/`: 로고, 이미지, 공통 CSS

## 운영 주의사항

- `review.html`은 내부 검수용이므로 정식 공개 전 삭제하거나 접근 경로를 별도로 관리할 수 있습니다.
- 문의 폼은 현재 화면 검수용입니다. 실제 전송을 위해서는 이메일, CRM 또는 폼 처리 서비스 연동이 필요합니다.
- 대표자, 회사 주소, 사업자 정보, 실제 연혁, 프로젝트 성과와 같은 항목은 최종 배포 전에 사실관계를 확인해야 합니다.
- 의료광고, 개인정보 수집, 외국인환자 유치 관련 문구는 최종 법무 검수를 권장합니다.
