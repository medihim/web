CRM·데이터

광고·상담·예약·매출 데이터를 연결해 더 나은 의사결정을 돕습니다.

현재 공개 채용 없음

WHY THIS ROLE

이 직무가
존재하는 이유.

고객이 어떤 콘텐츠와 광고를 통해 유입되고, 어떤 상담 과정을 거쳐 예약과 매출로 이어졌는지 하나의 데이터 흐름으로 연결합니다. 조직이 감이나 단편적인 지표가 아니라 동일한 정의와 신뢰할 수 있는 데이터로 판단하도록 돕습니다.

각 직무는 분리된 기능이 아니라 해외환자의 하나의 의료 여정을 중심으로 협업하는 전문 영역입니다.

RESPONSIBILITIES

주요 업무와 책임.

직무의 경계를 넘어 실제 고객과 운영의 문제를 해결하고, 반복 가능한 기준으로 남깁니다.

고객 여정과 상담 단계에 맞는 CRM 상태값과 데이터 항목을 설계합니다.
광고, 콘텐츠, 상담, 예약, 내원과 매출 데이터를 연결합니다.
UTM, 캠페인 코드와 고객 식별 기준을 관리합니다.
경영진과 실무 조직이 활용할 수 있는 대시보드와 리포트를 구축합니다.
데이터 누락, 중복과 불일치를 점검하고 품질을 관리합니다.
국가, 병원, 채널, 콘텐츠와 상담 단계별 전환 성과를 분석합니다.
반복 업무와 후속 상담을 자동화하는 워크플로우를 설계합니다.
고객정보의 접근권한, 보관기간과 개인정보 보호 기준을 관리합니다.

COLLABORATION

함께 일하는
방식.

글로벌 마케팅, 해외환자 상담·운영, 병원 운영·재무, 사업전략과 플랫폼·개발 조직과 협업합니다. 실무자가 실제로 입력하고 활용할 수 있는 데이터 구조를 함께 만들며, 지표의 명칭과 산출 기준을 문서화합니다. 데이터 오류가 발생하면 개인의 실수보다 입력·연동·운영구조의 원인을 먼저 확인합니다.

CAPABILITIES

필요한 역량과
우대 경험.

요구 역량

  • 사업과 운영의 질문을 데이터 항목과 지표로 전환하는 능력
  • 스프레드시트, CRM 또는 BI 도구를 활용한 데이터 관리 역량
  • 지표의 정의와 산출 기준을 명확하게 정리하는 능력
  • 데이터 오류의 원인을 추적하고 운영 가능한 해결책을 만드는 능력

우대 경험

  • CRM, RevOps, 마케팅 데이터 또는 영업운영 경험
  • GA4, GTM, UTM과 광고 플랫폼 데이터 운영 경험
  • Looker Studio 등 BI 및 시각화 도구 사용 경험
  • 병원·의료 데이터, 예약·결제 또는 매출 데이터 관리 경험
  • API 연동, 업무 자동화 또는 AI 기반 데이터 활용 경험

EXPECTED OUTCOMES

입사 후
기대하는 성과.

  • 기존 데이터의 출처, 정의, 누락과 중복 문제를 진단합니다.
  • 고객 식별 기준과 상담 단계, 핵심 데이터 필드를 표준화합니다.
  • 광고 유입부터 상담, 예약과 매출까지 연결된 퍼널 대시보드를 구축합니다.
  • 반복 리포트와 후속 상담 업무를 자동화해 운영 효율을 높입니다.

SCENE 05상담 상태에 따라 LINE 후속 메시지가 발송되는 자동화 흐름을 설계합니다.
